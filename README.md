# To-Do List Web App

This is a simple and user-friendly To-Do List Application built using *HTML, CSS and JavaScript*.  
Users can add and delete tasks easily. The app design is fully responsive and works smoothly on mobile and desktop devices.

---

##  Features
- Add new tasks
- Delete or mark completed tasks
- Clean and simple UI
- Mobile responsive using CSS Media Queries
- No page reload required

---

##  Technologies Used
- *HTML*
- *CSS*
- *JavaScript*
---

##  Responsive Design
Checked and tested on:
- Mobile devices
- Tablet view
- Desktop view  
(Used Chrome DevTools for testing responsiveness)
